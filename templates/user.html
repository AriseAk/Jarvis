User Manual for Jarvis

Welcome to your AI Chatbot! This guide will walk you through all the available features and how to use them effectively.

1. Starting a Chat

  • When you visit the homepage, the chatbot will greet you with a welcome message.
  • Type your query in the input box at the bottom and press the Send button.

2. Using Voice Input

  • Click the Mic button to use voice input.
  • The chatbot will record your speech for 5 seconds and transcribe it into text.
  • Once transcribed, press Send to submit.

3. Starting a New Chat

  • Click the New Chat button to start fresh.
  • This will clear the current chat history.

4. User Profile Options

  • Click on the Profile Icon in the sidebar to access options.
  • View your registered name and select:
    • Edit Personality: Customize chatbot responses.
    • Log Out: Sign out from your account.

5. Chat History

  • Past interactions are saved with timestamps.

6. AI Bot Capabilities

  • Assists with general queries, reminders, and voice commands.
  • Provides factual information and simple conversation.

7. Real-Time Responses

  • Time: Ask "What is the time?" for the current time.
  • Weather: Ask "What’s the weather like?" for live updates.
  • News: Ask "Tell me the latest news" for current events.

8. Entertainment Commands

  • Music & Videos: Use "Play a song" or "Play a video" to return an url.

9. Sending a Query

  • After typing or using voice input, press Send to submit.

10. System Requirements

  • Requires a web browser with an internet connection.
  • Ensure microphone permissions are enabled for voice input.

11. Troubleshooting

  • Check microphone settings if voice input doesn’t work.
  • Refresh the page if responses are delayed.
  • For login issues, verify credentials or reset your password.

Enjoy using your AI Chatbot!
